Usando o MonoliticoJS

O MonoliticoJS é um sistema web que realiza a verificação de equivalência forte entre programas monolíticos.

Criando um programa monolítico

Antes de poder realizar a verificação, é necessário criar os dois programas monolíticos. Para isso, basta clicar na opção "Criar programa" desejada (programa 1 ou programa 2). Em, seguida, será exibida a tela onde o programa monolítico será criado, instrução por instrução. Nessa mesma tela, é possível ainda:

  • Selecionar uma instrução já adicionada e editar as suas informações
  • Selecionar uma instrução e excluí-la do programa

Realizando a verificação

Tendo já os dois programas monolíticos cadastrados, é possível executar a verificação de equivalência forte entre eles. Para isso, basta clicar no botão VERIFICAR EQUIVALÊNCIA.
