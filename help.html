Usando o MonoliticoJS

O MonoliticoJS é um sistema web que realiza a verificação de equivalência forte entre programas monolíticos.

Criando um programa monolítico

A criação dos programas monolíticos é feita por instruções. Para começar, basta clicar na opção "Criar programa" desejada (programa 1 ou programa 2).

Inserindo uma instrução

Para inserir uma instrução no final do programa monolítico, primeiro selecione o tipo desejado: "Operação" ou "Condição". Em seguida, preencha os respectivos campos e clique no botão "Inserir"

Editando uma instrução

Após uma instrução ser inserida no programa, ela ainda pode ser alterada. Para isso, é necessário apenas clicar na linha desejada (os dados serão exibidos no painel à esquerda) e editar os valores. É possível inclusive alterar o tipo da instrução. Para finalizar, apenas clique em "Atualizar".

Excluindo uma instrução

Para excluir uma determinada instrução já adicionada ao programa, selecione-a na lista da direita e depois clique no botão "Excluir", no painel esquerdo.

Realizando a verificação

Tendo já os dois programas monolíticos cadastrados, é possível executar a verificação de equivalência forte entre eles. Para isso, basta clicar no botão VERIFICAR EQUIVALÊNCIA.

Analisando os resultados

Os resultados serão exibidos passo a passo abaixo da área utilizada para criar os programas monolíticos. O programa irá exibir automaticamente o resultado final. Para exibir o começo da verificação, basta utilizar o scroll do navegador para voltar ao início da página.

Para mais detalhes do funcionamento do programa, visite a página web do MonoliticoJS.
